Panduan Dasar Komponen UI

Belajar membuat elemen visual antarmuka website (UI) yang interaktif, modern, dan rapi untuk pemula.

👋 Selamat Datang!

Halaman ini dirancang khusus untuk pemula. Di sebelah kiri, Anda bisa mencoba berinteraksi langsung dengan komponen UI. Di sebelah kanan, terdapat kode HTML nya yang bisa Anda pelajari dan tiru.

🖱️ Buttons (Tombol)

Elemen interaktif utama dalam website. Gunakan warna berbeda untuk menunjukkan aksi utama (Primary) dan aksi pendukung (Secondary).

<!-- Tombol Utama (Primary) --> <button class="btn btn-primary"> Simpan Data </button> <!-- Tombol Garis Tepi (Outline) --> <button class="btn btn-outline"> Batalkan </button>

🃏 Cards (Kartu)

Kartu sangat bagus untuk mengelompokkan informasi. Sering digunakan untuk menampilkan artikel blog, produk, atau profil pengguna.

Belajar Koding

Memahami dasar-dasar HTML dan CSS untuk membangun tampilan antarmuka yang menarik.

<div class="demo-card"> <img src="image.jpeg" alt="Gambar"> <h3>Belajar Koding</h3> <p>Memahami dasar HTML & CSS.</p> <button class="btn btn-primary"> Mulai Belajar </button> </div>

📝 Input Form (Formulir)

Digunakan untuk mengambil data atau inputan dari pengguna. Desain form yang baik memiliki label yang jelas dan warna outline saat sedang diketik.

<div class="form-group"> <label for="nama">Nama Lengkap</label> <input type="text" id="nama" class="form-input" placeholder="Contoh: Budi" > </div>

🗂️ Navigasi Tabs

Berguna ketika Anda ingin menghemat tempat dengan membagi konten ke dalam beberapa bagian yang bisa di-klik ganti.

  • Profil
  • Keamanan
  • Notifikasi

Klik salah satu menu di atas untuk mencoba efek interaktifnya.

<ul class="demo-tabs"> <li class="active">Profil</li> <li>Keamanan</li> <li>Notifikasi</li> </ul> <!-- (Untuk interaksinya perlu --> <!-- sedikit Javascript) -->
